# paypal-nvp-api

Simple paypal nvp api integration

[![Build Status Travis](https://travis-ci.org/AmastyLtd/paypal-nvp-api.svg?branch=master)](https://travis-ci.org/AmastyLtd/paypal-nvp-api)


### Requirements:

- PHP >= 7.1.3
- ext-curl


### Installation:
```bash
composer require amasty/paypal-nvp-api
```
